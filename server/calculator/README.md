# Web calculator

This web-app sample implements a calculator for simple expressions such as
```code
3+4*5
5*9+(4*5)+3+(3*9)
y=x+5*x
```
## Running Locally

Make sure you have [Node.js](http://nodejs.org/)  installed.

```sh
npm install
npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).
